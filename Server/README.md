## Overview

This creates the server for the rebel challenge with the following API:

|API | Description | Request body | Response body |
|--- | ---- | ---- | ---- |
|`GET /api/artists` | Get all artists | None | Array of artists|
|`GET /api/artists/{id}` | Get an artist by ID | None | artist|
|`POST /api/artists` | Add a new artist | To-do item | artist |
|`PUT /api/artists/{id}` | Update an existing artist &nbsp; | artist | None |
|`DELETE /api/artists/{id}` &nbsp; &nbsp; | Delete an artist &nbsp; &nbsp; | None | None|
